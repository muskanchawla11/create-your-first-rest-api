# Express

From folder `1` to `12`, we built a REST API in Express, step by step.

## Lessons

1. **1-creating_server_using_express** — set up the Express app and listen on a port
2. **2-sending_a_response** — send responses with `res.send` / `res.json`
3. **3-serve_data** — serve static data via a GET route
4. **4-extracting_query_params_from_req_body** — read query parameters from the request
5. **5-filtering_by_query_params** — filter data based on query params
6. **6-path_params** — route parameters with `:id` style paths
7. **7-path_params_1** — more practice with path params
8. **8-path-params-2** — extending path-param handling
9. **9-expressRouter** — splitting routes into modular routers
10. **10-modulise_the_project** — modularising the project structure
11. **11-handling_route_not_found** — 404 handling for unknown routes
12. **12-CORS** — enabling CORS for cross-origin requests
